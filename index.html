Portfolio

  • ActionCable-Vue

    Opensource

    While working on a Rails + Vue client project that needed realtime multi-conversation chat, I realized just how tedious it was to work with ActionCable in Vue. This realization led to me writing a Vue plugin that made using ActionCable in Vue dead-easy and much more delightful to work with. ActionCable-Vue currently has over 2000 downloads on npm and 40 stars on Github.

  • eTrack

    Client - Edge Outsource

    eTrack is a fleet tracking application I worked on for Edge Outsource Ltd in collaboration with TroTro Tractor Ltd. It is an Android application that allows clients of Edge Outsource to primarily view the details (driver, registration number) and location of their fleet. It also features the ability to send engine commands to vehicles, and retrieve and playback vehicle location history data.

  • Peralta

    Client - Africa Foresight Group

    As part of a 6-week internship, I worked with a senior consultant to deliver Peralta. It's a Vue + Rails 5 app that currently powers talent management for Africa Foresight Group's talent pool of over 50 consultants. I spearheaded the engineering effort to migrate the application's frontend architecture from Rails MVC to Vue SPA.

  • Anyato.live

    Client - Nnanna Chioma

    Chioma is a friend. This is a personal website that I designed and developed with her while teaching her how to code. It features a Glitch Node.JS API I built that (was featured by Glitch) retrieves any Medium user's posts.

  • CEWP

    Client - CEWP

    CEWP is a project I worked on for the Energy Commission of Ghana as part of a campaign to get Ghanaians to use only Certified Electricians for electrical work. It lists the over 6000 Certified Electricians in Ghana and allows you to search for and contact those nearest to you. I built both the Android app and the ASP.NET WEB API backend that powers the location-aware electrician search feature.

  • Programming Ideas 2

    Personal

    When I began learning how to code I was always looking for ideas of apps to build in a bid to increase my proficiency so I built IdeaBag. It currently has over 150 ideas, 3 difficulty levels to cater to varying degrees of proficiency and a well-received user interface that I designed. IdeaBag currently has more than 130,000 installs and an enviable 4.6/5 rating after 500+ reviews in the Google Play Store.

  • Programming Ideas 2 (WEB)

    Personal

    Several users rightfully complained about IdeaBag being an Android-only application. In an effort to appease those complaints, I ported IdeaBag to the web with Vue.JS. After open-sourcing it, contributors have helped bring it to feature-parity with its Android counterpart.

  • Present, Sir

    Hackathon (2nd place) - forLOOP Ghana UG Hackathon

    Over the course of 24 hours at the maiden edition of forLOOP Ghana's hackathon, I developed "Present, Sir". It is a class USSD-based class attendance tracker built using Africa's Talking's USSD API. I built the Xamarin.Android based app and used ASP.NET MVC5 + LiteDB for the backend.
